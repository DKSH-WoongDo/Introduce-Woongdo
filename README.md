<h1 style="font-weight:600">API</h1>

<p>또한, 사이트 레이아웃은 사전 예고 없이 갑작스럽게 변경될 수 있으며 그럴 때마다 크롤러가 오작동할 수 있으므로, <b><i>공식적으로 지원하는 API를 사용하시는 것을 권장드립니다.</i></b></p>
